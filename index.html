About this landingpage

In this SPA you can dynamically create a section with a simple header and a little description. Then the link will be automatically added to the navbar. There is also present the smooth scrolling to each section.

Instruction:

  • Go to "create your section"
  • Insert the title of the section and the description
  • Submit the section

Notes:

If you try to submit another section with the same title it will give you a warning! You cannot create two sections with the same title but you can create two section with the same content
